# 🏏 Cricket Player Rankings Analysis using Web Scraping

## 📌 Project Overview
- This project focuses on extracting and analyzing cricket player rankings data from the CREX website using web scraping techniques.

- The dataset includes player rankings across different cricket formats such as ODI, T20, and Test for Batting, Bowling, and All-rounder categories.

- The project demonstrates end-to-end data analytics skills including web scraping, data cleaning, preprocessing, exploratory data analysis (EDA), and insight generation   using Python.

---

## 🎯 Objectives
- Extract cricket player rankings data from the CREX website using web scraping techniques
- Clean and preprocess the extracted ranking dataset
- Analyze player performance across formats and categories
- Compare rankings, ratings, and country-wise player representation
- Visualize insights using charts and graphs for better understanding of trends and comparisons

---

## 📂 Dataset Description
The dataset contains the following key attributes:

- Cricket Format (ODI, T20, Test)
- Rank
- Player Name
- Country
- Rating
- Category (Batting, Bowling, All-rounder)

The dataset includes 180 player ranking records collected from the CREX rankings website.

---

## 🛠️ Tools & Technologies Used
- Python
- Requests – sending HTTP requests to websites
- BeautifulSoup – web scraping and HTML parsing
- Pandas – data manipulation & preprocessing
- NumPy – numerical operations
- Matplotlib & Seaborn – data visualization
- Jupyter Notebook – analysis environment

---

## 🌐 Web Scraping Process
Key web scraping steps performed:

- Inspected the CREX rankings webpage structure using browser developer tools
- Retrieved webpage content using Python Requests library
- Parsed HTML content using BeautifulSoup
- Extracted ranking details using HTML tags, classes, and attributes
- Stored extracted data into structured Pandas DataFrames
- Exported cleaned data for analysis and visualization

---

## 🧹 Data Cleaning & Preprocessing
Key preprocessing steps performed:

- Checked and handled missing values
- Removed duplicate records
- Verified data consistency across ranking categories
- Converted ranking and rating columns into appropriate numerical data types
- Structured categorical columns for efficient analysis
- Prepared the dataset for visualization and statistical analysis

### Why Data Cleaning Was Important?
Data preprocessing helped to:

- Improve dataset reliability and consistency
- Ensure accurate statistical analysis
- Remove redundancy and inconsistencies
- Enable efficient visualization and comparison across formats and categories

---

## 📊 Exploratory Data Analysis (EDA)
The analysis includes:

- Country-wise player representation analysis
- Rating distribution analysis across categories
- Format-wise performance comparison
- Rank vs rating relationship analysis
- Country-wise performance heatmaps
- Distribution analysis using boxplots and histograms
- Category-wise performance comparison

---

## 🔍 Key Insights
- Batting and Bowling categories showed higher and more stable player ratings compared to All-rounders
- Test cricket players demonstrated the strongest overall rating consistency
- India had the highest number of ranked players in the dataset
- Sri Lanka recorded one of the highest average player ratings despite lower representation
- Player ratings decrease progressively with increasing rank position
- Australia and New Zealand showed balanced performance across Batting and Bowling categories
- Emerging cricket nations had comparatively lower rating distributions and player representation
